# RSVP - Fazendinha do Léo

Aplicação web para confirmação de presença de festa infantil.

## Tecnologias
- HTML5, CSS3, JavaScript (Vanilla)
- Google Sheets (Banco de Dados)
- Google Apps Script (Backend/E-mails)
- GitHub Pages (Hospedagem)

## Como utilizar
1. Configure o Google Sheets e o Apps Script conforme tutorial.
2. Substitua a constante `SCRIPT_URL` no `script.js` pela URL gerada no Apps Script.
3. Publique no GitHub Pages.
